CodeSynapse is a futuristic, AI-powered analytics dashboard designed for Computer Science students. It aggregates coding performance data from multiple platforms, visualizes skill competencies, and provides an AI Career Coach to generate personalized study roadmaps and resume reviews.

## 🚀 Key Features

### 1. 🔗 Multi-Platform Identity Uplink
* **Real-time Sync:** Fetches user profiles from **GitHub**, **LeetCode**, **HackerRank**, and **GeeksForGeeks**.
* **Data Aggregation:** Consolidates solving streaks, repository counts, and problem-solving stats into a single unified view.

### 2. 📊 Competency Radar Matrix
* **Visual Analytics:** Uses **Plotly Express** to generate interactive Cyberpunk-styled radar charts.
* **Skill Scoring:** Normalizes data from different platforms to calculate an overall "Competency Score" (0-100).

### 3. 🤖 AI Nexus Coach
* **Powered by Google Gemini:** A built-in AI chatbot contextually aware of your stats.
* **Career Advice:** Ask for resume reviews, project ideas, or DSA roadmaps based on your weak areas.
* **Self-Healing Logic:** Robust error handling ensures the AI always replies, even if the API hiccups.

### 4. 📄 Encrypted Career Report
* **Instant PDF Generation:** One-click download of a professional career summary.
* **ReportLab Integration:** Generates high-quality, printable PDFs on the fly.

## 🛠️ Tech Stack

* **Frontend:** [Gradio](https://gradio.app/) (Custom Cyberpunk Theme)
* **Backend:** Python 3.11, AsyncIO
* **AI Engine:** Google Gemini Pro API
* **Data Visualization:** Plotly, Pandas
* **Web Scraping:** Playwright, AIOHTTP
* **Deployment:** Docker, Hugging Face Spaces
